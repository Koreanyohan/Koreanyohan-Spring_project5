문자열객체 strings

  • aBC에 a가 포함되었는지 : [[ ${ #strings.contains('aBC', 'a') } ]]
  • aBC에 b로 시작되었는지 : [[ ${ #strings.startsWith('aBC', 'b') } ]]
  • aBC를 인덱스2~3범위 컷 : [[ ${ #strings.substring('aBC', 2, 3) } ]]
  • aBC에서 a를 b로 바꾸기 : [[ ${ #strings.replace('aBC', 'a', 'b') } ]]
  • aBC를 대문자로 변환 : [[ ${ #strings.toUpperCase('aBC') } ]]
  • aBC를 소문자로 변환 : [[ ${ #strings.toLowerCase('aBC') } ]]

숫자객체 numbers

  • 자리수만큼 0으로 채워서 출력 : [[ ${#numbers.formatInteger(1,5)} ]]
  • 패턴을 지정하여 출력 : [[ ${#numbers.formatInteger(100000, 3, 'POINT')} ]]
  • 화폐단위로 출력 : [[ ${#numbers.formatCurrency(100)} ]]
  • 퍼센트단위로 출력 : [[ ${#numbers.formatPercent(0.2,2,0)} ]]

날짜객체 temporals

  • 현재날짜 : [[ ${date} ]]
  • 현재날짜를 특정패턴으로 출력 : [[ ${#temporals.format(date, 'yyyy-MM-dd')} ]]
  • 현재날짜를 특정패턴으로 출력 : [[ ${#temporals.format(date, 'yyyy년MM월dd일')} ]]
  • 현재시간을 특정패턴으로 출력 : [[ ${#temporals.format(date, 'hh시mm분ss초')} ]]
  • 현재시간을 특정패턴으로 출력 : [[ ${#temporals.format(date, 'hh:mm:ss초')} ]]
  • 년 : [[ ${#temporals.year(date)} ]]
  • 월 : [[ ${#temporals.month(date)} ]]
  • 일 : [[ ${#temporals.day(date)} ]]
  • 시 : [[ ${#temporals.hour(date)} ]]
  • 분 : [[ ${#temporals.minute(date)} ]]
  • 초 : [[ ${#temporals.second(date)} ]]
